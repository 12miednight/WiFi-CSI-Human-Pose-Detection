## 💡 How It Works  

The app takes advantage of WiFi signals sent by your router or adapter. These signals change when they bounce off human bodies. The software captures this data as Channel State Information (CSI), which holds fine details about signal variations. It then uses deep learning to analyze these changes and estimate body positions. This method lets you sense human poses without any visible camera.
